# Portfolio — Joshua Arzu Lardy

Site portfolio personnel présentant mes compétences, services et réalisations dans le domaine du design, de la vidéo et du multimédia.

## Aperçu

Portfolio développé en HTML/CSS vanilla dans le cadre de ma formation en écriture multimédia à l'ISFSC. Il présente mes compétences en UI/UX, montage vidéo, illustration et création de contenu.

## Structure du projet

```
portfolio/
├── index.html
├── style.css
└── assets/
    ├── images/
    │   └── photoProfil.jpg
    └── video/
        └── (vidéos hébergées sur Google Drive)
```

## Sections

- **Hero** — Présentation et appel à l'action
- **About** — Parcours et compétences (UX, Premiere Pro, Illustrator, Vidéo)
- **Services** — UI/UX · Premiere Pro · Illustrator · Vidéo
- **Projects** — Réalisations récentes avec captures et vidéos
- **Contact** — Email, téléphone et localisation

## Technologies

- HTML5
- CSS3 (variables CSS, Grid, Flexbox)
- Google Fonts — Poppins
- Vidéos hébergées sur Google Drive

## Lancer le projet

Aucune dépendance, aucun build nécessaire. Ouvre simplement `index.html` dans un navigateur.

```bash
open index.html
```
